# based on [getnf](https://github.com/ronniedroid/getnf)

### Dependencies
- curl
- unzip
- fzf
- fontconfig
- jq

### Installation
```sh
git clone 'https://github.com/khonsaloh/nf-dl.git'
cd nf-dl
sudo make install
```

### Uninstallation
Inside the cloned repo:
```sh
sudo make uninstall
```
or just remove the script
